A simple plugin to showcase the word & character count and reading time.


== Description ==

This is very simple plugin to showcase your post's word count, character count and reading time.
After activation, it will automatically show the counts on all of your post. You can customize the heading, or also enable or disable any of the Stats. [See the LIVE perview](https://getwebbricks.com/blogs/)

Features include:

* Show the word count of the post
* Show the character count of the post
* Show the reading time
* Custom heading
* Change the position of the stats from top or to the bottom
* Translate this to any language
